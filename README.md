# Simple-Quiz-App
A simple command-line Quiz Application built using Python that presents multiple-choice questions, accepts user input, validates answers, calculates the final score, and provides instant feedback. This project demonstrates the use of lists, dictionaries, loops, conditional statements, and user interaction in Python.
## Overview
A Python-based command-line quiz application that asks multiple-choice questions, checks user answers, provides instant feedback, and displays the final score.

## Features
- Multiple-choice questions
- User-friendly command-line interface
- Instant feedback for each answer
- Automatic score calculation
- Final score summary

## Technologies Used
- Python 3

## Concepts Covered
- Lists and dictionaries
- Loops
- Conditional statements
- User input handling
- Basic program logic

## How to Run
1. Clone the repository.
2. Open a terminal in the project folder.
3. Run:
   ```bash
   python main.py
   ```
